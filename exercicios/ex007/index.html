Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> </b> (não semântica).

Nesta frase, temos um termo em destaque usando a tag <strong> </strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> </i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> </em> (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> </mark>

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo; usando as tags <big> </big> <small> </small>

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.
Usando a tag <del> </del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
Usando a tag <ins> </ins>

Exisre tambés o sublinhado com a tag <u> </u> (não semântica)

Texto sobrescrito

Para insetir coisas do tipo x20+3
Usando a tag <sup> </sup>

Texto subscrito

Para insetir coisas do tipo H2O
Usando a tag <sub> </sub>
